# Nea-Hub Dashboard

Dashboard web para el bot Discord RUPC.

## Contenido
- `dashboard.html` - Interfaz principal
- `dashboard.js` - Lógica del dashboard
- `assets/` - CSS y librerías

## Uso
Abre `index.html` en un navegador o accede en GitHub Pages.

## Seguridad
✅ No contiene código del bot
✅ No contiene credenciales
✅ Solo archivos estáticos públicos
